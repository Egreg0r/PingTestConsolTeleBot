Телеграмм бот

- Список команд:
  /ping - пингует указанный ip.
  /membercount - кол-во подключенных пользователей к боту. 
- Бот переодически пингует сервера указанные в списке <Servers>  файла Xml.config.
- Все настройки по управлению ботом хранятся в фале "Xml.Config"
- "Log4Net.config" отвечает за настройки логов работы бота. (Изменяйте с осторожностью.)
